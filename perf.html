Things To Do Before Filing For a Performance Issue in CloudHealth

  • We assume that you are using Google Chrome. If not, please install Google Chrome first before proceeding.
  • When capturing a screenshot, make sure that the error details are exposed:
    • For Windows: Press F12
    • For Mac: Press Command + Shift + C
  • Make sure that the console is docked at the bottom to expose the most lines of code:
  • Go to the "Network" tab, and sort the calls according to "Time". Make sure the sort is ascending so that we see at the bottom the slowest operation in that particular frame.
